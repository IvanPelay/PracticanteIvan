# Base de Datos – Generador de Reportes Automatizados

Este proyecto contempla dos implementaciones de base de datos en SQL Server:

## v1_sin_plantillas
Versión inicial del sistema.
- Plantillas definidas directamente en BD
- Sin precarga de reportes
- Menor dependencia de integraciones externas

## v2_con_plantillas_svn
Versión extendida del sistema.
- Plantillas externas (SVN)
- Precarga de configuraciones y ejemplos
- Integración con Paperless y múltiples BD
- Cambia el flujo de ejecución del sistema
